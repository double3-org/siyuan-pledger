你好，欢迎使用 pLedger 思源账本, 一个基于思源笔记的账本插件

## 用前必读

- **插件配置**
    - **任何配置均不要频繁变更，变更前建议切换【数据存放位置】或备份原始数据，数据丢失是不可逆的**
    - 【**数据存放位置**】必须为文档ID，不能是笔记本ID
    - 【**配置**】中推荐配置的数据必须只能是两层，目前插件不支持多层级的复杂配置
    - 【AI配置】是非必要的配置，如果不配置，则无法使用远程AI 服务

- **插件工作方式之资产部分**
    - 插件会在你配置的【数据存放位置】文档下按年创建新的子文档，用于存储资产数据
    - 你在文档中的修改会影响插件，如果你不知道你在做什么，请不要在文档中直接修改资产数据

- **插件工作方式之记账部分**
    - 不定时更新，敬请期待

## 待改进

从产品的角度，我知道这个插件有很多待改进的地方：
- 数据统计部分不够流畅，尤其是跨年的时候会有明显的割裂
- AI 部分不够智能，只能进行简单的图片识别，缺少一些必要 skill
- 不支持历史数据的编辑，目前只能通过编辑文档来间接实现
- 不支持多层级的复杂配置，自由度较低
- 存在影响思源本体的样式的问题，例如可能会改变滚动条的样式
- 对移动端的支持不足，操作起来不够丝滑
- 不支持多语言，目前仅支持中文
- 缺少足够的引导

不过，考虑到该需求比较小众，当前版本已经基本满足我对该插件的想象
如果未来有其他朋友使用，我会考虑在后续版本中进行优化
