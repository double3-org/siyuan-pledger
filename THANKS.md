# 致谢（Acknowledgements）

本项目 **siyuan-pledger** 的实现离不开以下优秀的开源项目与其贡献者。在此一并致以诚挚的感谢。

## 依赖（Dependencies）

- **Vue**  
  渐进式 JavaScript 框架，为项目提供响应式 UI 能力 
  [去看看](https://github.com/vuejs/vue) 

- **Tailwind CSS**  
   CSS 框架，用于构建一致且高效的界面样式
   [去看看](https://github.com/tailwindlabs/tailwindcss) 

- **DaisyUI**  
  基于 Tailwind CSS 的组件库
   [去看看](https://github.com/saadeghi/daisyui) 

- **ECharts**  
  功能强大的可视化图表库，用于数据展示
   [去看看](https://github.com/apache/echarts) 

- **cally**  
  轻量级日期选择组件
   [去看看](https://github.com/WickyNilliams/cally) 

- **currency.js**  
  用于安全、精确地处理货币与数值计算
   [去看看](https://github.com/scurker/currency.js) 

- **Vite**  
  下一代前端构建工具，提供极速的开发体验
   [去看看](https://github.com/vitejs/vite) 

- **其他** 
  `@vitejs/plugin-vue`、`TypeScript`、`vue-tsc`、`ts-node`、`@types/node`、`cross-env`、`autoprefixer`、`fast-glob`、`minimist`、`rollup-plugin-livereload`、`vite-plugin-static-copy`、`vite-plugin-zip-pack` 等构建工具
