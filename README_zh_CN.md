### 写在前面
本项目基于插件模板项目 [plugin-sample-vite](https://github.com/frostime/plugin-sample-vite/tree/main) 创建, 移除了 `make-link` 等相关 `scripts` 文件, 使用 [siyuan-plugin-cli](https://github.com/frostime/siyuan-plugin-cli/tree/main) 进行软连接和版本号管理等操作。


### 如何启动项目
`npm run dev` 启动项目

`npx make-link-win` 创建软连接 **OR** `npx make-install` 复制文件到思源工作区 

`npx check-link` 检查状态
